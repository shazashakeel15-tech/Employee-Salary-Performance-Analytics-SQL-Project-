Employee Salary & Performance Analytics (SQL Project)
Overview

This project analyzes employee salary data across departments using SQL.
It focuses on salary distribution, employee performance ranking, and payroll contribution analysis.

Tools Used
SQL
Window Functions
CTEs
Joins
Key Analyses:

Employee master dataset creation using JOINs
Department-level salary KPIs
Employee ranking within departments
Top performers identification
Salary fairness analysis
Payroll contribution percentage calculation

SQL Concepts Applied:

JOINs
CTEs
Window Functions (RANK, DENSE_RANK, SUM OVER, AVG OVER)
LAG / LEAD
CASE WHEN
Aggregations

Key Insights:

Identified top earning employees per department
Found salary distribution differences across departments
Measured employee contribution to total department payroll
Analyzed salary fairness using department averages
Performed trend analysis using LAG/LEAD
